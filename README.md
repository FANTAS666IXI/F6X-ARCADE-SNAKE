# F6X ARCADE SNAKE
- Description :

  - Arcade Snake game like in Unity.

- Controls :

  - Move Up         : W / ↑
  - Move Left       : A / ←
  - Move Down       : S / ↓
  - Move Right      : D / →
  - Pick Items      : Touch It
  - Increase Volume : + / NumPad +
  - Decrease Volume : - / NumPad -
  - Exit game       : Escape

- Versions :

  - F6X ARCADE SNAKE : 0.4.0
  - Unity            : 2022.3.8f1
